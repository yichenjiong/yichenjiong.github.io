####001
#####关于print

  • 可以输出运算后的数字
  • 可以输出拼接及重复的字符串
  • 输出拼接+不可以拼接不同类型

####002
注意缩进

BIF:built - in function内置函数

#####BIF函数信息的获取
dir(_builtins)获取内置函数列表

help（内置函数名）获取内置函数用法帮助

###003变量与字符串

####变量

  • 变量需要赋值才能使用
  • 变量名不可用数字开头，和c差不多
  • 区分大小写
  • =是赋值符号
  • 变量名尽量有意义
    ####字符串
  • 是为引号内的内容
  • +对字符串的作用是拼接
  • 引号可以是‘’也可以是“”但一定要成对
  • 字符串内的引号打印可以利用转义字符\也可单双引号交替使用
    #####原始字符串
  • 使用方法是在字符串引号前面加一个r如r‘C:\now’打印出来的字符是不会进行转义的
  • 转义字符串末尾不能直接加反斜杠\

###004条件分支

####if else 语句
格式：

if 条件 ：
    条件为真（True）执行的操作
else：
    条件为假执行的操作

####while循环
格式：

while 条件 ：
       条件为真执行的操作

####逻辑操作符and
and可以将任意表达式连接到一起并返回一个布尔类型的值
####随机数生成:random模块
在使用模块之前必须导入模块
如导入random模块语句如下

import random

#####randint()函数
会返回一个随机的整数
