003变量与字符串

变量

  • 变量需要赋值才能使用
  • 变量名不可用数字开头，和c差不多
  • 区分大小写
  • =是赋值符号
  • 变量名尽量有意义

    字符串

  • 概念：是为引号内的内容
  • +对字符串的作用是拼接
  • 引号可以是‘’也可以是“”但一定要成对
  • 字符串内的引号打印可以利用转义字符\也可单双引号交替使用
    原始字符串
  • 使用方法是在字符串引号前面加一个r如r‘C:\now’打印出来的字符是不会进行转义的
  • 转义字符串末尾不能直接加反斜杠\

005 python的数据类型

数值类型
  • 整型：即整数
    • 长整型
    • 整型 python3中这俩合并了
  • 浮点型：带小数点则为浮点型，没有则为整型
  • e记法，即科学计数法，这个e不区分大小写
    • e记法会自动变成浮点型
    • 15e10代表15*10的10次方
  • 布尔类型
    • 是为特殊的整数类型
    • 有true和fale两种类型，默认为1和零
    • 所以可以用于计算
    • 不过最好不要用于计算
  • 转化函数
    • int（）将数据类型转化为整型
      • 括号中的必须为数字型的字符串，或者浮点数
      • 对浮点数的操作是直接取整
    • float（）
      • 将整数或数值型字符串转换为浮点数
    • str（）
      • 是将一个数或任何其他类型转换为一个字符串
  • python的BIF名可以当作变量名来使用，但被当作变量名使用的BIF会失去其原函数功能
获取关于类型的信息
  • type（）函数，可返回数据类型
  • isinstance（a,b）a为待确定类型的数据，b是指定的数据类型
    该函数返回一个布尔类型的值，当a为指定数据类型时返回true，否则返回false

006常用操作符

加减乘除 +-/*

  • +=之类的类似c语言中的用法在python中也可以用

  • 赋值符连着使用也是可以的，例如a = b = c = d = 10相当于给abcd都赋值10

  • 除法是真正的除法，不是整数则转换为浮点数

  • 求余%

  • 幂运算操作符**例如3**5的意思是35

  • 地板除法 //
    除法结果是小数的话舍弃小数，整数取法是去相邻整数中较小的整数，例如10//8为1，-8//3 = -3

  • 基本就是先乘除后加减，括号内先算

  • 关于幂运算操作符的优先级
    幂运算操作符的优先级：对于一元操作符，比其左侧运算的优先级高，比右侧的优先级高

逻辑操作符 and or not

都是一元操作符，功能略

3<4and4<5

007 8 9分支和循环

if else

c语言中if-else-if-else结构在python中也有

形式是：

if 条件：
    执行段
elif 条件：
    执行段
else：
    执行段
三元操作符

例如：
small = x if x<y else y

这行代码，当x<y时将x赋值给small否则将y赋值给small

断言 assert

应用形式：例如 assert 3>4

作用：当关键字assert后面的条件为假得到时候，程序自动崩溃并抛出AssertionError异常，一般用于在程序中插入检查点，用在需要确保程序中某个条件一定为真才能让程序执行下去的情况

    for 目标 in 表达式:

    循环体

    注：目标是一个变量，而表达式是一个列表或元组或字符串

      range([start,] stop[,step=1])
