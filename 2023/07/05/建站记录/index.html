准备

首先按照 hexo 官网上给定的教程下载依赖的一系列东西，然后选择一个文件夹
在这个文件夹里 右键 git Bash here 如下图所示

然后 在这个文件中安装管理hexo文件的工具，安装hexo 初始化hexo
命令如下：

npm install -g hexo-cli

npm install hexo

hexo init

然后这个文件夹下边就会有hexo必要的文件
